# Code Quiz

## Description:
The coding quiz is an app to test your knowledge of JavaScript. Click start and use your coding skills to answer each question. If you answer incorrectly your time/score will be deducted by 10 seconds. 

## Technologies used: 
* HTML
* CSS
* Javascript
